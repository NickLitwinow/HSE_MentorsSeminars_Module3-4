# Домашнее задание №3: MinIO + JupyterHub

## Структура проекта

```
Homework 3/
├── docker-compose.yml   # Конфигурация для запуска MinIO
├── hw1_minio.ipynb      # Jupyter ноутбук с выполнением задания
└── README.md            # Этот файл
```

## Быстрый старт

### 1. Запуск MinIO

```bash
# Перейти в папку с заданием
cd "Homework 3"

# Запустить MinIO
docker-compose up -d
```

### 2. Проверка работы MinIO

- **Веб-интерфейс (Console):** http://localhost:9001
- **API Endpoint:** http://localhost:9000
- **Логин:** `minioadmin`
- **Пароль:** `minioadmin123`

### 3. Выполнение ноутбука

Откройте `hw1_minio.ipynb` в JupyterHub и выполните все ячейки последовательно.

## Что реализовано в ноутбуке

1. ✅ Подключение к MinIO через Python-клиент
2. ✅ Проверка соединения (список бакетов)
3. ✅ Создание бакета `hw-litvinov`
4. ✅ Создание тестового DataFrame
5. ✅ Сохранение данных в формате CSV
6. ✅ Сохранение данных в формате Parquet
7. ✅ Скачивание и чтение данных обратно
8. ✅ Проверка целостности данных

## Используемые технологии

- **MinIO** — S3-совместимое объектное хранилище
- **Python** — язык программирования
- **minio** — официальный Python SDK для MinIO
- **pandas** — библиотека для работы с данными
- **pyarrow** — для поддержки формата Parquet

## Остановка MinIO

```bash
docker-compose down
```

Для полного удаления данных:

```bash
docker-compose down -v
```
